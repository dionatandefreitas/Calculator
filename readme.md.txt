Esta é uma calculadora simples, desenvolvida com HTML, CSS e JavaScript. Ela permite ao usuário realizar operações básicas de adição, subtração, multiplicação e divisão.

Tecnologias Utilizadas
HTML
CSS
JavaScript
Instruções de Uso
Abra o arquivo HTML da calculadora em seu navegador web.
Insira dois valores na calculadora.
Escolha a operação desejada (adição, subtração, multiplicação ou divisão).
Clique em "=" para obter o resultado.
